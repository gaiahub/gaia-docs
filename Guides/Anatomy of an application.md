## Anatomy of an application

<br>
![Application image|400x400](images/gaia_hero_nodes.png)
#### What's an app?
An application is a set of nodes connected together and interact with OpenAI or other alternative. 


#### What's a node?
Nodes are in the core of an applicatin. Nodes are like black boxes that execute actions. Nodes can connect to a data base, 
do a cvs file, or can connect other nodes togheter and create an app.

#### What's a credential?
Some services that nodes interact with will ask for a API key sometimes. So you need to import those API keys into GaiaHub's credentials.
